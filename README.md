#### FFMPEG 学习代码

* SDL0.cpp 使用SDL的基本程序，显示了一个窗口
* FFMPEG0.CPP 使用FFMPEG进行视频基本的基本过程，提取视频的前5帧，参考 [dranger教程](http://dranger.com/ffmpeg/tutorial01.html)

* FFmpeg1.cpp 将解码后的图像使用SDL2.0输出。
